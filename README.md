
A premium academic cover page designer with live A4 preview, gold-themed styling, watermark logo support, and one-click PDF export using jsPDF.
A premium web-based tool for designing professional academic cover pages with live preview and direct PDF export functionality. This project allows students to quickly generate elegant, university-style cover pages for assignments, lab reports, projects, and thesis submissions.

# ✨ Features

📄 Real A4 Live Preview (210mm × 297mm layout)

🎨 Elegant Gold-Themed Design

🏛 Logo Upload with Watermark Support

🖼 Automatic Background Watermark Effect

📝 Dynamic Form Input → Live Update

📅 Date Selection

📥 One-Click PDF Download

⚡ Built with HTML, CSS, JavaScript & jsPDF

🌐 Fully Client-Side (No Backend Required)



 # 🛠 Technologies Used

HTML5

CSS3 (Custom Gold Theme Styling)

JavaScript (DOM Manipulation)

jsPDF Library

html2canvas (via jsPDF html rendering)

# 🚀 How to Use

Enter university and course details

Upload institution logo

Fill student and instructor information

Preview the design live

Click “Download PDF”

Your professional cover page will be generated instantly.
